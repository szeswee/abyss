# abyss

A WebGL/[three.js](https://www.threejs.org/) topographical generator inspired by Duffy London's [*Abyss Table*](https://www.duffylondon.com/product/tables/abyss-table/).

**Demo:** https://codepen.io/ardecila/pen/ZajyBN
